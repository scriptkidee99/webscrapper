This is a project representing a web scrapper website

This is just a prototype

For running this, do the following things first

1. You will find a requirments.txt file in the same forlder where you are reading this file

2. open terminal in the same folder

3. type command - pip install -r requirments.txt

4. This will install all the requirments for the project

5. Now start the server by using - python manage.py runserver
